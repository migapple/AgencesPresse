# News Agencies

# 1.2
We've enhanced the app with several interactive features to address this feedback:

An interactive 3-round quiz (country, founding date, legal status), with selectable rounds, real-time scoring, and a final results screen, generating up to 15 randomized questions per session from 42 agencies.
An interactive map (Leaflet/OpenStreetMap) showing each agency's headquarters, with direct navigation from each detail screen.
Continent-based filtering and real-time search across the database.
For each agency: two distinct facts, legal status, broadcast languages, and a direct link to the official website (available for 37 of the 42 agencies).
A "My Apps" section for discovering the rest of the catalog, organized by category with filters.

The app now offers a complete educational experience combining a searchable database, interactive map, quiz, and category browsing, going well beyond a static list.
